# Архив вспомогательных файлов

Эта папка содержит вспомогательные скрипты и файлы, которые не требуются для основного функционирования парсера.

## Содержимое

### Скрипты проверки и верификации
- Скрипты для проверки отчетов
- Скрипты для сравнения данных с реальными страницами
- Скрипты для верификации результатов парсинга

### Временные файлы
- Временные скрипты разработки
- Тестовые файлы
- Старые версии парсеров

### Скрипты туннелей
- Скрипты для настройки Cloudflare Tunnel
- Скрипты для других туннельных решений

### PowerShell скрипты
- Вспомогательные скрипты для мониторинга
- Скрипты для логирования

## Примечание

Эти файлы сохранены для справки, но не используются в основном рабочем процессе парсера.
