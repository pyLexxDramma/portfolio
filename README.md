# Парсер Яндекс.Карты и 2ГИС

Веб-приложение для автоматического сбора статистики о компаниях из Яндекс.Карт и 2ГИС.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python run_server.py
```

Приложение будет доступно по адресу: http://localhost:8000

## Функционал

- Поиск компаний по названию и сайту
- Сбор статистики с Яндекс.Карт и 2ГИС
- Формирование отчетов в PDF
- Email уведомления
- Прогресс-бар с этапами парсинга
